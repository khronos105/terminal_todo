# Terminal TODO App

**Version 1.0.0**

Simple TODO List in terminal made with NodeJS

---

## Objectives

- Create task
- List tasks
- List tasks by state(completed/pending)
- Toggle completed tasks
- Delete task
- Persist tasks into a json file

---

## Installation

- npm install
- node app.js
